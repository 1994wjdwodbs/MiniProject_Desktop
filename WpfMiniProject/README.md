## 🎦WpfMiniProject
네이버와 구글 API를 이용한 영화 검색 프로그램입니다.</br>

네이버 API를 이용하여 네이버 사이트에 등록된 영화 정보를 제공하고</br> 
검색한 네이버 영화 정보를 재가공하여 구글 Youtube API에 질의합니다.</br>
그러면 해당 API에서 관련된 내용(섬네일, 제목, 주소...)들을 리스트로 만들어</br>
사용자들이 보기 쉽도록 출력합니다.</br>
